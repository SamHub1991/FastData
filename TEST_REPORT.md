### 资源使用趋势

```
同步前:
  sqlserver   572 MiB  (37.27%)
  mysql       360 MiB  (70.36%)
  postgres    32 MiB   (12.40%)

  sqlserver   573 MiB  (37.30%)  +1 MiB
  mysql       361 MiB  (70.56%)  +1 MiB
  postgres    33 MiB   (12.81%)  +1 MiB
```

**分析**: 同步过程资源消耗极低，适合长时间稳定运行。
